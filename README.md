# Projeto de Melhoria - Grupo Evolução

## Descrição

Esta página web apresenta o projeto de melhoria desenvolvido pelo **Grupo Evolução** para a linha de montagem 407, com foco na organização e segurança no armazenamento dos grampos de fixação do teto.

O projeto aborda um problema de falta de padronização, onde os grampos, após o uso, eram deixados em locais inadequados, gerando desorganização, riscos de acidentes e perda de tempo. A solução implementada envolveu a criação de um sistema de armazenamento e transporte eficiente, resultando em ganhos significativos de produtividade e segurança.

## Tecnologias Utilizadas

A página foi desenvolvida utilizando as seguintes tecnologias:

*   **HTML5:** para a estruturação do conteúdo.
*   **CSS3:** para a estilização da página, incluindo um layout responsivo.
*   **JavaScript:** para a interatividade, como a galeria de imagens (Lightbox).

## Desafios no Desenvolvimento

Durante a criação da página, alguns desafios foram encontrados:

*   **Estruturação do Conteúdo:** Organizar as informações de forma clara e lógica, seguindo a narrativa do projeto de melhoria (contexto, análise, solução e resultados).
*   **Responsividade:** Garantir que a página fosse visualmente agradável e funcional em diferentes tamanhos de tela (desktops, tablets e smartphones).
*   **Integração de Mídia:** Incorporar imagens e vídeos de forma otimizada para não comprometer o tempo de carregamento da página.

## Como Executar o Projeto

Para visualizar a página, basta abrir o arquivo `index.html` em qualquer navegador web.
